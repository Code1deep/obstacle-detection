# Détection des obstacles cognitifs

Algorithmes de détection et de classification des obstacles
cognitifs (linéarité, temporalité, proportionnalité) à partir
des traces d'apprentissage.

Développé dans le cadre d'une recherche doctorale en didactique
des systèmes complexes (Design-Based Research).

## Statut

Prototype de recherche — Cycle 1 DBR.
